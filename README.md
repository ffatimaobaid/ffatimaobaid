### Featured Projects

**[HRFlux](https://github.com/ffatimaobaid/hrflux)** — Multi-agent HR automation system
Built with LangGraph, FastAPI, ChromaDB, and React. Four coordinated agents (LeaveBot, PolicyBot, DocuBot, EscalationBot) handle HR workflows end-to-end using hybrid RAG over company policy docs, powered by Groq Llama-3.3-70b. Shortlisted by VR Digital LLP at FAST's Projects & Career Expo.

**[Genesis Voice Agent](https://github.com/ffatimaobaid/genesis-voice-agent)** — RAG-powered voice assistant for car listings
A voice-based RAG agent that scrapes live car listings (Playwright), indexes them with sentence-transformer embeddings in ChromaDB, and answers spoken buyer queries using Whisper (speech-to-text), Groq Llama-3.3-70b (generation), and ElevenLabs (voice output). Built as a technical assessment.

**Real-Time Emotion Recognition** — CNN-based facial emotion detection
Trained on FER-2013 with live webcam face tracking, served through a Flask interface for real-time inference.

**Multilingual Story-to-Audio Converter** — Emotion-aware TTS pipeline
Converts written stories into expressive narrated audio across Urdu, Roman Urdu, and English.
